MIUI v5 - MIUIAndroid XML Strings Korean Translations.
=================
This is repository for MIUIAndroid MultiRom project.

이곳은 MIUIAndroid의 다국어 롬을 위한 저장소입니다.

매주 빌드되는 MIUIAndroid MultiRom은 이 저장소로부터 파일을 가져갑니다.
